# IoT-LIFT-INDIA

Arduino programs made as part of a value-added course on IoT at college. Each file is a small standalone project.

## Files

- **7Segment.ino** — Displays digits 0–9 on a 7-segment display, one after another.
- **Brige.ino** — Controls two DC motors to open and close a bridge model.
- **EnergyMotor.ino** — Switches a motor on and off in a simple loop.
- **TrafficSignal.ino** — Runs a red-yellow-green traffic light sequence using LEDs.

## Components Used

- Arduino Uno/Nano
- 7-segment display
- DC motors + motor driver (for the bridge)
- LEDs (red, yellow, green)
- Jumper wires and breadboard

## How to Run

1. Open the `.ino` file you want in the Arduino IDE.
2. Connect the components as per the pin numbers in the code.
3. Select the correct board and port, then upload.
4. The circuit will start running automatically.
